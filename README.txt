Quick install
-------------

Just point your browser to the folder/location where you installed the application
and an installation wizard will be launched.

Problems with install
---------------------
If you have problem with install please let us know here
http://diagramo.com/contact.php or read INSTALL.txt


Details
---------
Bellow overview of the application.

If you want to read how to install application read INSTALL.txt
If you want to read the license read LICENSE.txt
If you want to know more about Diagramo copyright read COPYRIGHT.txt


This folder contains:

[/] - root folder contains the presentation site
   |--[assets] - css, images and othe look & feel files
   |--[common] - all the PHP files that makes the server side of the application
   |--[diagrams] - the place where all the diagrams will be stored
   |--[exporter] - the SVG to PNG exporter
   |--[lib] - all the JavaScripts of the web app engine
   |__README.txt - this file
   |__other files that belong to the application


This folder contains:

[/] - this folder
   |--[documents] - all documents related to diagramo: design, specs, sql, etc
   |--[web] - the Diagramo script itself
   |__README.txt - this file

To read find more read /web/README.txt file. I contains more specific information.
